# Free2Move Car Rental App

**Android Studio | Java & Kotlin | Firebase**

Free2Move is a mobile app for car rental management. Users can browse cars, make bookings, and track rentals in real time. The app uses **Firebase** for authentication and database management, and is built with **Java/Kotlin** in Android Studio.

## Features
- User login & registration
- Browse and book cars
- Real-time booking updates
- Admin panel for managing cars

## Tech Stack
- Java
- Kotlin
- Android Studio
- Firebase
